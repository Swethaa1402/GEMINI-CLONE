# Gemini Clone (AI Chatbot UI)

📌 Description

The Gemini Clone is a frontend-based AI chatbot interface that simulates interaction with an AI model. It provides a smooth and responsive chat experience where users can send queries and receive AI-generated responses dynamically.

🚀 Features

💬 Chat Functionality

* Chat interface with user and AI message display
* Messages aligned (user → right, AI → left)
* Timestamp for each message
* Enter key support for quick message sending
* Auto-clear input field after sending

🤖 AI Interaction

* Sends user input to AI API using JavaScript (fetch/XMLHttpRequest)
* Dynamically displays AI-generated responses
* Handles API errors with user-friendly messages 

⏳ Loading & UX Enhancements

* Animated loading indicator while waiting for response
* Disables input during processing to prevent multiple requests
* Auto-scroll to latest message

📋 Copy Feature

* Copy button to copy AI responses
* Clipboard support with confirmation tooltip (“Copied!”) 

🌗 Theme Customization

* Light/Dark mode toggle
* Theme preference saved using local storage
* Persistent theme even after page reload 

🧠 Chat History Management

* Stores chat history using browser local storage
* Displays previous conversations
* Option to delete entire chat history
* Confirmation prompt before deletion
* Empty state message when no chats exist 

📱 Responsive Design

* Works on desktop, tablet, and mobile devices
* Adaptive chat layout and scalable UI

🧩 Modular Design

* Landing Page Module (intro + start chat)
* Chat Interface Module
* Messaging Module (API communication)
* Session Management Module (local storage handling) 

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

▶️ How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
